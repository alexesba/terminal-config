# terminal-emulators/

Tracked templates for terminal emulator configs. `install.sh` copies the chosen
`.example` file into `~/.config/` (real file, not a symlink) so Gogh and other
tools can write theme data alongside your config.

| Template | Installed to |
|---|---|
| `alacritty.yml.example` | `~/.config/alacritty/alacritty.yml` |
| `kitty.conf.example` | `~/.config/kitty/kitty.conf` |
| `wezterm.lua.example` | `~/.config/wezterm/wezterm.lua` |

WezTerm also uses `~/.config/wezterm/colors.lua` (generated by `colorscheme`).

Run `./update.sh` to migrate legacy dotfiles symlinks to local copies.
